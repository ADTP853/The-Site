HTML5: Is the latest version of Hyper Text Markup language. Essentially, HTML is the spine of your web site, the part that keeps it all together. HTML allows developers to tell the computer what they want to show and when. Sort of like a translator.

CSS3: Is the latest version of Cascading Style Sheets. CSS is the difference between a site looking plain black and white, to a site looking bright and full of colour.

JS: JavaScript is still fairly new to the coding world. It allows your site to be interactive. For example, you can create a an email subscription list.

JQ: JQuery isn't a language a such. It's a programme built to support JS and as such, allows your page to load faster and run more smoothly. So it's a nice to have.
